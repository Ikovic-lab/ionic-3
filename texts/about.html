Where to get books?

Many sites provide e-books in EPUB format, without DRM protection. Some of my favorites are:

  • Project Gutenberg
  • epubBooks.com
  • O'Reilly
  • Penguin Books
  • Kobo

How to add books to Ionic?

  1. Download books with the N9 web browser, or copy them to the phone via USB, into the documents folder (MyDocs)
  2. Start Ionic, go to the library, and add books using the + icon
